Modul 3: Passwort-Challenge

Wie schnell wird dein Passwort geknackt? Password Game live + 3 goldene Regeln.

⏱
15 Min.
Dauer
👥
10–18 J.
Altersgruppe
🎯
Teilnehmende verstehen, warum starke Passwörter wichtig sind, und können selbst sichere Passwörter erstellen.
Lernziel
Password Game 2FA Passwort-Manager

Material: Beamer mit Internet, neal.fun/password-game, security.org/how-secure-is-my-password

Ablauf – Schritt für Schritt

Moderationstext zum Vorlesen – direkt einsetzbar.

1

Schock-Einstieg (2 Min.)

Am Beamer security.org/how-secure-is-my-password öffnen:

• «123456» → sofort geknackt ❌
• «Fussball2026» → 3 Stunden ⚠️
• «Käs3!Kuchen#Blau99» → 3 Millionen Jahre ✅

«Der Unterschied? Länge + Gross/Klein + Zahlen + Sonderzeichen + für jeden Account ein anderes!»

2

Password Game (8 Min.)

neal.fun/password-game am Beamer öffnen. Gemeinsam spielen – Klasse ruft Lösungen.

Die Regeln werden immer verrückter (Mondphase? Schachzug?). Es wird laut – das ist gewollt!

Ziel: Regel 10–15 erreichen.

Pointe: «Seht ihr, wie kompliziert ein WIRKLICH sicheres Passwort ist? Deshalb gibt es Passwort-Manager.»

3

3 Goldene Regeln (5 Min.)

Am Beamer:

🥇 Passwort-Manager installieren → Bitwarden (kostenlos, open-source)
🥈 Für die 3 wichtigsten Accounts (Insta, E-Mail, Schule) JETZT neues Passwort setzen
🥉 2FA aktivieren – überall, wo es geht

«Diese 3 Dinge könnt ihr heute Abend in 10 Minuten erledigen.»

🔗 Tools & Links

neal.fun/password-game security.org/how-secure-is-my-password bitwarden.com

👥 Altersanpassungen

🟢 Jüngere anpassen

10–13 J.: Nur Schock-Einstieg + 3 Regeln. Password Game ist für alle lustig.

🔴 Ältere anpassen

14–18 J.: Brute-Force-Angriffe erklären. Warum Passwort-Manager wichtig sind.
